  • Ctrl + S - Save file
  • Ctrl + X - Cut selected, delete entire line
  • Ctrl + C - Copy selected
  • Ctrl + V - Paste from Clipboard
  • Ctrl + B - Show/Hide Side Bar
  • Ctrl + A - Select all
  • Alt + ↑ or ↓ - Move current or selected line(s) up or down
  • Alt + Shift + ↑ or ↓ - Duplicate current or selected line(s) up or down
